# SweetMusicPlayer
功能：  
1，按照歌曲，专辑，艺术家显示相应信息  
2，加载歌曲列表（可以实现a-z字母检索）  
3，播放本地、在线音乐（MediaPlayer应用，AIDL运用）  
4，智能匹配本地歌词,在线歌词
